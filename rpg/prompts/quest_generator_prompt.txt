### ЗАДАНИЕ
1.  **Выбери одного NPC** из списка жителей текущей локации, который будет квестодателем.
2.  **Придумай для него ОДИН НОВЫЙ квест**, который логично вытекает из его профессии, контекста мира и НЕ повторяет идеи уже существующих квестов.
3.  **Опиши сцену выдачи квеста** от лица этого NPC.

### ТРЕБОВАНИЯ К ОТВЕТУ
Твой ответ должен быть ТОЛЬКО валидным JSON-объектом со следующей структурой:

{
  "quest_giver_name": "Имя выбранного NPC",
  "dialogue_line": "Прямая речь NPC, в которой он выдает квест. Например: 'Путешественник, у меня к тебе просьба. Мой караван был атакован гоблинами к северу отсюда...'",
  "quest": {
    "id": "уникальный_id_квеста",
    "name": "Название квеста",
    "description": "Описание квеста для журнала.",
    "objectives": [
      {
        "text": "Описание цели для игрока",
        "type": "тип_цели",
        "target_poi_id": "id_целевой_локации"
      }
    ]
  }
}

Доступные типы (`type`) целей: "reach_location", "obtain_item", "report_to_npc".
При выборе целей используй `id` локаций и предметов из контекста мира.
